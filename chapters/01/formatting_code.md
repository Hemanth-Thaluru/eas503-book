# Formatting Code

If a single statement becomes too big, you can have it span multiple lines by either surrounded the statement by parenthesis or using the line-continuation character, which is a \.

```{code-cell} ipython3
(2 +
3)
```

```{code-cell} ipython3
2 + \
3
```
